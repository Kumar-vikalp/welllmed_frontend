1# Medicine E-Commerce React App
2
3A modern, animated, and interactive medicine marketplace built with React, Tailwind CSS, Framer Motion, Lottie, and React Router v6+. UI-only, no backend required.
4
5## Features
6
7- Auth pages (Login, Signup)
8- Home page with product grid, filters, skeleton loaders
9- Product details with carousel, related products
10- Cart, Checkout, Payment, Profile, Order History
11- Responsive, animated, and beautiful UI
12- Context API for cart and user state
13- Dummy fetch APIs (local JSON/Axios stubs)
14- Toast notifications, skeletons, Lottie animations
15
16## Tech Stack
17
18- React 19+
19- Tailwind CSS
20- Framer Motion
21- Lottie React
22- React Router v6+
23- Axios
24
25## Getting Started
26
271. Install dependencies: `npm install`
